## Usage
The PSR embed component can be customised using data attributes. 
 
```jsx
    <VsContainer>
        <VsRow>
            <VsCol md="6">
                <vs-psr-embed
                    :config="[{'lang': 'fr'},{'subSearchType': 'acco'},{'locplace': '4161'}]"
                />
            </VsCol>
        </VsRow>
    </VsContainer>
```
